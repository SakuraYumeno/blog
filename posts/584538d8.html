Butterfly 主题美化配置

自定义文件结构

+--- css
|   +--- custom.styl
|   +--- _custom
|   |   +--- background.css
|   |   +--- font.css
+--- font
|   +--- CascadiaMono.woff2
|   +--- ZhuZiAWan.woff2
+--- img
|   +--- avatar.jpg
+--- js

在 /blog_root/source 中创建 css/font/js 目录，在 css 目录中创建 _custom 用于存放自定义 css 文件。
创建 custom.styl 文件，添加如下内容:

@import '_custom/*.css'

修改主题配置文件，添加 inject 参数

# Inject
# Insert the code to head (before '</head>' tag) and the bottom (before '</body>' tag)
# 插入代码到头部 </head> 之前 和 底部 </body> 之前
inject:
  head:
    - <link rel="stylesheet" href="/blog/css/custom.css" media="defer" onload="this.media='all'">
  bottom:
    # - <script src="xxxx"></script>

设置主题背景

一图流

修改主题配置文件_config.butterfly.yml

default_top_img: transparent
background: url(/blog/img/background.jpg)
footer_bg: transparent

渐变色

  1. 修改主题配置文件_config.butterfly.yml
background: "#efefef"
  1. 创建自定义 css 文件
/* 文章页背景 */
.layout > div:first-child:not(.recent-posts) {
  /* 以下代表白色透明度为 0.3 */
  background: rgba(255, 255, 255, 0.3);
}
/* 所有背景（包括首页卡片、文章页、页面页等） */
#recent-posts > .recent-post-item,
.layout > div:first-child:not(.recent-posts),
.layout_post > #page,
.layout_post > #post,
.read-mode .layout_post > #post {
  /* 以下代表白色透明度为 0.3 */
  background: rgba(255, 255, 255, 0.3);
}
/* 侧栏卡片 */
.card-widget{
  background:rgba(255, 255, 255, 0.3)!important;
}

/* 日间模式页脚字体颜色 */
[data-theme="light"]{
  #footer{
    background: rgba(255, 255, 255, 0.3)!important;
    /* 页脚 a 标签字体颜色 */
    #footer-wrap {
      color: #050505;

      color: #050505;

  }
}

中英文字符间加空格

安装插件

npm install hexo-filter-auto-spacing --save

配置文件 _config.yml 添加参数

# hexo 中英文之间自动加空格 
auto_spacing:
  enable: true
