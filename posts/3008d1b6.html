Shrinking XFS root partition

  1. Reboot the system in Rescue Mode by booting Installation Media

    Select Rescue a Oracle Linux system.

    Select option 3 - Skip to shell.

  2. Preparing a backup by xfs_dump

[root@localhost ~]# lvs --segments -o +devices
  LV      VG      Attr       #Str Type   SSize  Devices
  data_lv data_vg -wi-------    1 linear 10.00g /dev/sdb1(0)

  home    rhel    -wi-a-----    1 linear 26.99g /dev/sda2(512)
  root    rhel    -wi-a-----    1 linear 50.00g /dev/sda2(7422)

[root@localhost ~]# vgchange -a y rhel

[root@localhost ~]# mkdir /mnt/root
[root@localhost ~]# mkdir /mnt/home
[root@localhost ~]# mount /dev/rhel/root /mnt/root/
[root@localhost ~]# mount /dev/rhel/home /mnt/home/

  1. reduce and format root logical volume, restore the root content
[root@localhost ~]# umount /mnt/root

[root@localhost ~]# lvreduce -L -5G /dev/rhel/root

  THIS MAY DESTROY YOUR DATA (filesystem etc.)
Do you really want to reduce rhel/root? [y/n]: y

  Logical volume rhel/root successfully resized.
[root@localhost ~]# vgreduce rhel /dev/sdd1
