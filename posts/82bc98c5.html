Block Browser and EDitor

发表于2023-05-20|更新于2023-06-09|DatabaseOracleUtilities
|阅读量:
  1. 编译 bbed
[oracle@centos7 ~]$ cd $ORACLE_HOME/rdbms/lib
[oracle@centos7 lib]$ make -f ins_rdbms.mk $ORACLE_HOME/rdbms/lib/bbed
[oracle@centos7 lib]$ ./bbed
Password: blockedit

BBED: Release 2.0.0.0.0 - Limited Production on Tue Mar 8 14:35:08 2022

Copyright (c) 1982, 2007, Oracle.  All rights reserved.

************* !!! For Oracle Internal Use only !!! ***************

BBED>
  1. 设置参数

查询数据文件信息，并保存至 bbed filelist 文件中 

SQL> set heading off
SQL> set feedback off
SQL> spool /u01/bbedlist.parf
SQL> select file#||''||name||' '||bytes from v$datafile;
1 /u01/app/oracle/oradata/orcl/system01.dbf 461373440
2 /u01/app/oracle/oradata/orcl/undotbs01.dbf 414187520
3 /u01/app/oracle/oradata/orcl/sysaux01.dbf 272629760
4 /u01/app/oracle/oradata/orcl/users01.dbf 5242880
5 /u01/app/oracle/oradata/ccdata.dbf 1825570816
SQL> spool off;

创建 parameter file

cat >> /u01/bbed.par <<EOF
blocksize=8192
listfile=/u01/bbedlist.parf
mode=edit
EOF
  1. 连接 bbed
[oracle@centos7 lib]$ ./bbed parfile=/u01/bbed.par password=blockedit
