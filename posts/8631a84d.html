Patch 6880880 - OPatch for DB 19.0.0.0.0

p6880880_190000_Linux-x86-64

Update OPatch Utility

To update Opatch, use the following instructions:

  1. Download the OPatch utility to a temporary directory.
  2. For each Oracle RAC database home and the GI home that are being patched, run the following commands as the home owner to extract the OPatch utility.
    $ unzip <OPATCH-ZIP> -d <ORACLE_HOME>
$ <ORACLE_HOME>/OPatch/opatch version

Record 1: Update OPatch utility version to 12.2.0.1.32 for Single Instance

Check the current opatch version.

[root@oracle19cdg ~]# /u01/app/oracle/product/19.0.0/dbhome_1/OPatch/opatch version
OPatch Version: 12.2.0.1.17
OPatch succeeded.

Take a backup of ORACLE_HOME/OPatch.

[root@oracle19cdg ~]# mv /u01/app/oracle/product/19.0.0/dbhome_1/OPatch{,.bak.`date +%m%d`}

Unzip the OPatch downloaded zip into ORACLE_HOME directory.

[root@oracle19cdg ~]# unzip -q /soft/p6880880_190000_Linux-x86-64.zip -d /u01/app/oracle/product/19.0.0/dbhome_1/
[root@oracle19cdg ~]# chown -R oracle:oinstall /u01/app/oracle/product/19.0.0/dbhome_1/OPatch

Check the version of the opatch utility on each node.

[root@oracle19cdg ~]# ls -ld /u01/app/oracle/product/19.0.0/dbhome_1/OPatch
drwxr-x--- 15 oracle oinstall 4096 Jul  4  2022 /u01/app/oracle/product/19.0.0/dbhome_1/OPatch
[root@oracle19cdg ~]#
[root@oracle19cdg ~]# /u01/app/oracle/product/19.0.0/dbhome_1/OPatch/opatch version
OPatch Version: 12.2.0.1.32
OPatch succeeded.

Record 2: Update OPatch utility version to 12.2.0.1.32 for RAC

Check the current opatch version on each node.

[root@oracle19crac ~]# /u01/app/19.0.0/grid/OPatch/opatch version
OPatch Version: 12.2.0.1.17
OPatch succeeded.
[root@oracle19crac ~]# /u01/app/oracle/product/19.0.0/dbhome_1/OPatch/opatch version
OPatch Version: 12.2.0.1.17
OPatch succeeded.

Take a backup of ORACLE_HOME/OPatch on each node.

[root@oracle19crac ~]# mv /u01/app/19.0.0/grid/OPatch{,.bak.`date +%m%d`}
[root@oracle19crac ~]# mv /u01/app/oracle/product/19.0.0/dbhome_1/OPatch{,.bak.`date +%m%d`}

Unzip the OPatch downloaded zip into ORACLE_HOME directory on each node.

[root@oracle19crac ~]# unzip -q /soft/p6880880_190000_Linux-x86-64.zip -d /u01/app/19.0.0/grid/
[root@oracle19crac ~]# chown -R grid:oinstall /u01/app/19.0.0/grid/OPatch
[root@oracle19crac ~]# unzip -q /soft/p6880880_190000_Linux-x86-64.zip -d /u01/app/oracle/product/19.0.0/dbhome_1/
[root@oracle19crac ~]# chown -R oracle:oinstall /u01/app/oracle/product/19.0.0/dbhome_1/OPatch

Check the version of the opatch utility on each node.

[root@oracle19crac ~]# ls -ld /u01/app/19.0.0/grid/OPatch
drwxr-x--- 15 grid oinstall 4096 Jul  4  2022 /u01/app/19.0.0/grid/OPatch
[root@oracle19crac ~]# ls -ld /u01/app/oracle/product/19.0.0/dbhome_1/OPatch
drwxr-x--- 15 oracle oinstall 4096 Jul  4  2022 /u01/app/oracle/product/19.0.0/dbhome_1/OPatch
[root@oracle19crac ~]#
[root@oracle19crac ~]# /u01/app/19.0.0/grid/OPatch/opatch version
OPatch Version: 12.2.0.1.32
OPatch succeeded.
[root@oracle19crac ~]# /u01/app/oracle/product/19.0.0/dbhome_1/OPatch/opatch version
OPatch Version: 12.2.0.1.32
OPatch succeeded.
