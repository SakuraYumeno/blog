在线扫描新添加的硬盘：

添加一块 SCSI 类型的硬盘，执行下面命令扫描磁盘

EMC PowerPath 环境中增加 LUN：

  1. 确保存储端已配置好 LUN 并映射给 Linux 主机，通过 powermt 命令获得 HBA 卡列表，并扫描列表中所有的 HBA 卡

  2. 执行 powermt config 配置新认到的磁盘，生成虚拟设备

  3. 执行 powermt display 检查新识别到的 LUN 及设备路径

  4. 执行 powermt save 保存
